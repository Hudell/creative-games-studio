Modify Tiled Map

To modify the map itself, download and use the Tiled Map Editor.
Please note: Both Tiled and this window modify the same file, so if you change the map here you have to reload the map there.
If you make any changes to the map on Tiled, make sure to press the "Reload Map" button bellow so that you don't lose anything.
